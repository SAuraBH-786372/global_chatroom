About Global Chatroom

Hi, I'm Saurabh Joshi, the creator of this chatroom.

This chat platform was built using React.js for the frontend and Node.js with WebSockets for real-time communication.

It enables users to connect globally in a seamless and interactive way without storing any messages.

The tech stack used in this project includes:

  • Frontend: React.js, CSS
  • Backend: Node.js, WebSockets
  • Deployment: Render

Thank you for visiting! Hope you enjoy using this chatroom.
